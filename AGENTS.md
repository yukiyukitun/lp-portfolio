# AGENTS.md

このプロジェクトは、ポートフォリオ掲載用のLP制作プロジェクトです。

## 制作テーマ

女性向けヨガスタジオ体験レッスンLP

## 最重要方針

Image2で作成したLPデザイン画像を、そのままページ全体に貼り付けないでください。

参考画像は、あくまでデザインの方向性を確認するための資料です。

必ず HTML / CSS / Tailwind CSS の構造として再現してください。

## 実装で再現するもの

参考画像から、以下を読み取って実装してください。

- レイアウト
- セクション構成
- 余白
- 配色
- フォントサイズ
- 文字の強弱
- CTAボタン
- 背景装飾
- カードデザイン
- 全体の雰囲気
- スクロールしたくなる流れ

## 実装で禁止すること

以下は禁止です。

- 参考画像をそのまま1枚画像として貼り付けること
- LP全体を画像だけで構成すること
- 文字を画像化すること
- 読みにくい小さすぎる文字にすること
- 余白を詰めすぎること
- ボタンが目立たないデザインにすること
- スマホで崩れるレイアウトにすること
- デザイン画像と無関係な派手な装飾を追加すること

## 技術方針

使用技術は以下を想定します。

- Next.js
- TypeScript
- Tailwind CSS

## デザイン方針

全体の印象は、上品で清潔感があり、やさしく安心できる女性向けLPにしてください。

目指す雰囲気：

- 上品
- 清潔感
- やさしい
- 安心感
- 女性向け
- 余白を活かす
- 初心者でも申し込みやすい
- 強すぎない
- 売り込み感が強すぎない

## カラー方針

以下の色を中心に使ってください。

- 白
- ペールグリーン
- ベージュ
- くすみピンク

色は淡く上品に使い、全体が安っぽくならないようにしてください。

## ターゲット

25〜45歳の女性。

想定ユーザー：

- ヨガ初心者
- 運動が苦手な人
- 忙しい毎日の中で自分の時間を作りたい人
- 心と体を整えたい人
- 肩こりや姿勢の悩みがある人
- 強い運動より、無理なく続けられる習慣を求めている人

## LPの目的

LINE登録から体験レッスン予約へつなげること。

そのため、CTAは分かりやすく、安心して押せるデザインにしてください。

主なCTA文言の例：

- LINEで体験レッスンを予約する
- まずは体験レッスンに申し込む
- LINEで空き状況を確認する

## セクション構成の目安

LPは以下の流れを基本にしてください。

1. ファーストビュー
2. 共感・悩み
3. スタジオの特徴
4. 体験レッスンの内容
5. 選ばれる理由
6. お客様の声
7. 料金・キャンペーン
8. よくある質問
9. 最終CTA

必要に応じて調整して構いませんが、申し込みまでの流れが自然になるようにしてください。

## 動き・アニメーション方針

静止画をそのまま再現するだけでなく、Webサイトとして自然な動きを入れてください。

ただし、派手すぎる動きは禁止です。

推奨する動き：

- ファーストビューのやわらかいフェードイン
- CTAボタンの軽いホバーアニメーション
- セクションごとの自然な表示
- カードの軽い浮き上がり
- 背景装飾の控えめな動き
- スクロール時に心地よく見える余白設計

避ける動き：

- 激しいアニメーション
- 点滅
- 過剰なパララックス
- 読み込みが重くなる演出
- ヨガLPに合わない派手な動き

## レスポンシブ方針

PC、タブレット、スマホで自然に見えるようにしてください。

特にスマホでは以下を重視してください。

- 文字が読みやすい
- CTAボタンが押しやすい
- 余白が詰まりすぎない
- 画像やカードが崩れない
- 横スクロールが発生しない

## コード方針

コードは分かりやすく、後から修正しやすい形にしてください。

意識すること：

- コンポーネントを適度に分ける
- classNameを整理する
- 同じようなUIは再利用しやすくする
- 不要なコードを残さない
- buildエラーが出ない状態にする
- TypeScriptの型エラーを残さない

## 画像の扱い

参考画像は `references` フォルダに保存します。

参考画像はデザイン確認用です。

LP内で画像を使う場合も、ページ全体を1枚画像として貼るのではなく、必要な部分だけを適切に使用してください。

## 最終的な完成イメージ

ポートフォリオに掲載したときに、以下が伝わる完成度を目指します。

- 女性向けLPが作れること
- 余白や配色のバランスを考えられること
- CTA導線を設計できること
- AI画像を参考にしながらWeb実装に落とし込めること
- Next.js / TypeScript / Tailwind CSSでLPを構築できること
- Codexで品質調整まで行えること
