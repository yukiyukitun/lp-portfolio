# project-setup.md

## このファイルの目的

このファイルは、LP制作環境 `lp-portfolio` の作業手順と初期設定をまとめるためのメモです。

このプロジェクトでは、Image2で作成したLPデザイン画像を参考にしながら、Claude Code CLIとCodexを使って、Next.js / TypeScript / Tailwind CSSでLPを実装します。

最終的には、ポートフォリオ掲載できる品質のLPを制作することを目的としています。
