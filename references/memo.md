# references memo

このフォルダには、Image2で作成したLPデザイン画像を保存します。

画像は、LP実装時の参考資料として使用します。

## 重要ルール

保存した画像を、そのままLP全体に貼り付けることは禁止です。

画像はあくまで参考として使用し、実装では HTML / CSS / Tailwind CSS の構造として再現します。

## 今回の制作テーマ

女性向けヨガスタジオ体験レッスンLP

## LPの目的

LINE登録から体験レッスン予約へつなげること。

## ターゲット

25〜45歳の女性。

- ヨガ初心者
- 運動が苦手な人
- 忙しい毎日の中で心と体を整えたい人
- 自分のための時間を作りたい人
- 安心して通えるヨガスタジオを探している人

## デザイン方向性

上品で清潔感があり、やさしく安心感のある女性向けLPにする。

重視する雰囲気：

- 上品
- 清潔感
- やさしい
- 安心感
- 余白
- 女性らしさ
- 初心者でも申し込みやすい空気感

使用したい色：

- 白
- ペールグリーン
- ベージュ
- くすみピンク

## 実装時に特に見るポイント

### 1. ファーストビュー

確認すること：

- メインコピーの位置
- サブコピーの見せ方
- CTAボタンの目立たせ方
- 写真や装飾の配置
- 余白の広さ
- 第一印象のやさしさ

### 2. 配色

確認すること：

- 白の使い方
- ペールグリーンの使い方
- ベージュの使い方
- くすみピンクの使い方
- 背景色と文字色のコントラスト

### 3. セクション構成

確認すること：

- 悩みへの共感
- スタジオの特徴
- 体験レッスンの内容
- 選ばれる理由
- お客様の声
- 料金やキャンペーン
- よくある質問
- 最終CTA

### 4. CTA

確認すること：

- LINE登録へ誘導できているか
- ボタンが分かりやすいか
- 押したくなる安心感があるか
- しつこすぎないか
- 複数箇所に自然に配置できているか

### 5. Web実装として調整するポイント

参考画像を見ながら、Webサイトとして以下を調整する。

- PCで見やすい横幅
- スマホで読みやすい文字サイズ
- タップしやすいボタンサイズ
- スクロールしやすいセクション間の余白
- 動きのある自然な表示
- 画像に頼りすぎない構造
- HTMLとして意味のある見出し構造

## Claude Code CLI に伝えるときの注意

Claude Code CLIには、以下を必ず伝えます。

- 参考画像をそのまま貼らない
- HTML / CSS / Tailwind CSSで構造として再現する
- Next.js / TypeScript / Tailwind CSSで実装する
- 女性向けヨガLPとして上品で清潔感のあるデザインにする
- CTAはLINE登録を目的にする
- 動きは控えめで自然に入れる

## Codex に伝えるときの注意

Codexには、以下を重点的に依頼します。

- 余白の調整
- レスポンシブ調整
- CTAの見やすさ改善
- hoverやfadeなどの自然な動きの追加
- コードの整理
- buildエラーの修正
- TypeScriptエラーの修正
- ポートフォリオ掲載に耐える完成度への調整
