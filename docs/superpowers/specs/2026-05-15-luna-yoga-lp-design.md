---

## 1. プロジェクト概要

### 目的
LINE登録から体験レッスン予約へつなげること。

### ターゲット
25〜45歳の女性。ヨガ初心者・運動が苦手な人・忙しい毎日の中で心と体を整えたい人。

### スタジオ名
Luna Yoga

### 技術スタック
- Next.js（App Router）
- TypeScript
- Tailwind CSS（カスタムカラー設定あり）
- Google Fonts: Shippori Mincho（見出し）+ Noto Sans JP（本文）


## 5. セクション仕様

### 5-1. Header
- 固定ヘッダー（スクロール後は白背景＋シャドウ）
- 左: ロゴ「Luna Yoga」（Shippori Mincho, yoga-cta色）
- 右: CTAボタン「体験レッスンを予約する」

### 5-2. HeroSection（ファーストビュー）
**目的**: 最初の数秒で内容が伝わること。CTAが見つけやすいこと。

- **背景**: ペールグリーン（`yoga-green`）→ 白のグラデーション
- **レイアウト**: PC=2カラム（左:テキスト / 右:ビジュアル）、SP=縦積み
- **右側ビジュアル**: 背景グラデーション＋丸型フレーム（CSSのみ、プレースホルダー色块）
- **コピー**:
  - メイン: 「整うだけじゃない。キレイもついてくるヨガ。」
  - サブ: 「初心者歓迎。少人数制で、心と体をやさしく整える体験レッスン。」
- **CTA**: 「LINEで体験レッスンを予約する」（LINE緑ボタン）
- **キャンペーン**: 「LINE登録で体験レッスン500円OFF」バッジ
- **アニメーション**: `fadeInUp`（0.6s ease）

### 5-3. ConcernSection（悩み・共感）
**目的**: ターゲットの悩みに共感し、「自分のことだ」と感じさせる。

- **背景**: `yoga-beige`
- **見出し**: 「こんなお悩み、ありませんか？」
- **内容**: アイコン＋テキストのカード、3〜4点
  - 最近、体が重くなってきた
  - 運動を始めたいけれど続くか不安
  - 忙しくて自分の時間が取れない
  - 肩こりや姿勢が気になっている
- **レイアウト**: PC=2列グリッド、SP=1列

### 5-4. FeaturesSection（スタジオの特徴）
**目的**: スタジオの魅力を視覚的に整理する。

- **背景**: 白
- **見出し**: 「Luna Yogaの特徴」
- **内容**: 4カード（アイコン＋タイトル＋説明）
  - 初心者歓迎・丁寧な指導
  - 少人数制（定員6名）
  - 女性専用の落ち着いた空間
  - 駅徒歩3分・LINEで簡単予約
- **レイアウト**: PC=4列、SP=2列
- **カード**: 角丸、シャドウ、ホバーで軽い浮き上がり

### 5-5. TrialSection（体験レッスンの内容）
**目的**: 申し込み前の不安を取り除く。

- **背景**: `yoga-green`（薄め）
- **見出し**: 「体験レッスンのご案内」
- **内容（左カラム）**:
  - 所要時間: 60分
  - 持ち物: 動きやすい服装のみ（ヨガマット無料貸出）
  - 参加資格: 初めての方大歓迎
  - 予約方法: LINEから簡単予約
- **右カラム**: CTAボタン＋キャンペーン文言
- **レイアウト**: PC=2カラム、SP=縦積み

### 5-6. ReasonsSection（選ばれる理由）
**目的**: 他スタジオと差別化する信頼感を伝える。

- **背景**: `yoga-base`
- **見出し**: 「Luna Yogaが選ばれる理由」
- **内容**: 番号付き3〜4点
  - 01. 全員が初心者からスタートできる安心の指導
  - 02. 少人数制だから自分のペースで続けられる
  - 03. 忙しい方のための夜・土日クラスも充実
  - 04. 体験後の強引な勧誘は一切なし
- **レイアウト**: PC=横並び番号カード、SP=縦積み

### 5-7. TestimonialsSection（お客様の声）
**目的**: 実際の声で信頼感・安心感を高める。

- **背景**: `yoga-pink`（薄め）
- **見出し**: 「ご参加された方のお声」
- **内容**: 架空の声3件
  - Aさん（30代）: 「初めてでも先生が丁寧に教えてくださって、すごく安心できました。」
  - Bさん（40代）: 「少人数なので質問しやすく、自分のペースで続けられています。」
  - Cさん（20代）: 「仕事帰りに通えるので、続けやすいです。体が軽くなりました。」
- **レイアウト**: PC=3列カード、SP=1列
- **デザイン**: 引用符（"）装飾、カード白背景

### 5-8. PricingSection（料金・キャンペーン）
**目的**: 料金への不安を解消し、申し込みを後押しする。

- **背景**: 白
- **見出し**: 「料金・キャンペーン」
- **体験レッスン料金**: ~~2,500円~~ → **2,000円**（LINE登録特典500円OFF）
- **通常レッスン**: 月4回 8,000円〜
- **CTAバナー**: 「LINE登録で体験レッスン500円OFF」（`yoga-cta`背景）

### 5-9. FaqSection（よくある質問）
**目的**: よくある疑問を事前に解消する。

- **背景**: `yoga-base`
- **見出し**: 「よくあるご質問」
- **内容**: 5問（アコーディオン形式、`useState`）
  - Q: 運動が苦手でも参加できますか？
  - Q: ヨガマットや道具は必要ですか？
  - Q: どんな服装で来ればいいですか？
  - Q: 体験レッスンの予約はどうすればいいですか？
  - Q: 体験後に入会しなければいけませんか？

### 5-10. FinalCtaSection（最終CTA）
**目的**: ページを読んだ人の背中を最後に押す。

- **背景**: `yoga-green`グラデーション
- **見出し**: 「まずは、体験レッスンから。」
- **サブコピー**: 「無理なく、あなたのペースで始められます。LINEから簡単に予約できます。」
- **CTA**: 「LINEで体験レッスンを予約する」（大きめ・LINE緑）
- **キャンペーン再掲**: 「LINE登録で体験レッスン500円OFF」

### 5-11. Footer
- ロゴ、コピーライト（© 2024 Luna Yoga）
- スタジオ情報（架空）: 東京都渋谷区〇〇1-2-3 / 営業時間など

## 6. 共通コンポーネント

### CtaButton
```typescript
interface CtaButtonProps {
  label: string;
  href?: string;
  size?: 'sm' | 'md' | 'lg';
  variant?: 'line' | 'outline';
}
```
- `line`: LINE緑背景、白テキスト
- `outline`: 白背景、yoga-cta色のボーダー＋テキスト
- ホバー: `opacity-90` + `translateY(-2px)` + `shadow-lg`

## 7. アニメーション（globals.css）

```css
@keyframes fadeInUp {
  from { opacity: 0; transform: translateY(20px); }
  to   { opacity: 1; transform: translateY(0); }
}
.animate-fadeInUp {
  animation: fadeInUp 0.6s ease forwards;
}
```

各セクションの最初の要素に `.animate-fadeInUp` を付与。

## 8. レスポンシブ方針

| ブレークポイント | 対応 |
|---|---|
| SP（〜767px） | 1カラム、フォント縮小、余白調整 |
| タブレット（768〜1023px） | 2カラム |
| PC（1024px〜） | フルレイアウト |

- 横スクロール禁止: `overflow-x: hidden` を body に設定
- CTAボタン: SP時は `width: 100%` でタップしやすく
