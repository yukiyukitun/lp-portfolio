## 作品タイトル

女性向けヨガスタジオ体験レッスンLP

## 架空クライアント設定

女性向けの少人数制ヨガスタジオを運営する架空クライアントを想定したLP制作です。

初心者でも安心して参加できる体験レッスンへの申し込みを増やすため、LINE登録を主な導線としたランディングページを制作しました。

## 制作目的

LINE登録から体験レッスン予約へつなげることを目的に制作しました。

ヨガ初心者や運動が苦手な方でも安心して申し込めるよう、強い売り込み感を避け、やさしく信頼感のある導線設計を意識しています。

## ターゲット

25〜45歳の女性。

主な想定ユーザー：

- ヨガ初心者
- 運動が苦手な方
- 忙しい毎日の中で自分の時間を作りたい方
- 心と体を整えたい方
- 肩こりや姿勢の悩みがある方
- 安心して通える少人数制スタジオを探している方

## デザインコンセプト

### コンセプト

やさしく整う、私のためのヨガ時間。

### 説明

白、ペールグリーン、ベージュ、くすみピンクを基調に、清潔感と安心感のある女性向けLPとして設計しました。

余白を広めに取り、情報を詰め込みすぎず、初めてヨガに参加する方でも不安なく読み進められる構成にしています。

CTAはLINE登録を主目的とし、ファーストビュー・中間・最終セクションに自然に配置することで、申し込みまでの導線を分かりやすくしました。

## デザインで意識したこと

- 初心者でも安心できるやさしい雰囲気
- 女性向けらしい上品で清潔感のある配色
- 売り込み感を抑えた自然なCTA導線
- 余白を活かした読みやすいレイアウト
- スマホでも見やすい情報整理
- 体験レッスンの内容が分かりやすい構成
- LINE登録への導線が迷わない設計

## 実装で意識したこと

- 参考画像をそのまま貼り付けず、HTML / CSS / Tailwind CSSで構造化
- Next.js / TypeScript / Tailwind CSSで実装
- セクションごとにコンポーネントを整理
- PC、タブレット、スマホに対応
- CTAボタンの視認性と押しやすさを調整
- hoverやfadeなどの自然な動きを追加
- buildエラーやTypeScriptエラーが出ないように調整
