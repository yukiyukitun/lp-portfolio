# Codex Prompt

このファイルは、CodexにLPの仕上げ調整を依頼するためのプロンプトです。

## 依頼内容

女性向けヨガスタジオ体験レッスンLPの仕上げ調整をしてください。

このLPは、Image2で作成した参考画像をもとに、Claude Code CLIで Next.js / TypeScript / Tailwind CSS として実装したものです。

Codexでは、初回実装を大きく作り直すのではなく、ポートフォリオ掲載に耐える完成度へ整えてください。

## 最初に確認してほしいこと

まず、プロジェクト直下の `AGENTS.md` を読んでください。

その方針に従って、現在の実装を確認し、必要な修正を行ってください。

## 最重要ルール

参考画像を、そのままページ全体に貼り付けることは禁止です。

LP全体を画像として表示するのではなく、HTML / CSS / Tailwind CSS の構造として実装されている状態を維持してください。

もし画像をベタ貼りしている箇所があれば、構造化されたHTML / CSS / Tailwindの実装に置き換えてください。

## 今回のLPテーマ

女性向けヨガスタジオ体験レッスンLP

## LPの目的

LINE登録から体験レッスン予約へつなげること。

CTAはLINE登録を主目的にしてください。

主なCTA文言：

- LINEで体験レッスンを予約する
- LINEで空き状況を確認する
- まずは体験レッスンに申し込む

## ターゲット

25〜45歳の女性。

想定ユーザー：

- ヨガ初心者
- 運動が苦手な人
- 忙しい毎日の中で自分の時間を作りたい人
- 心と体を整えたい人
- 肩こりや姿勢の悩みがある人
- 安心して通える少人数制のヨガスタジオを探している人

## デザイン方向性

上品で清潔感があり、やさしく安心感のある女性向けLPにしてください。

雰囲気：

- 上品
- 清潔感
- やさしい
- 安心感
- 余白を活かす
- 女性向け
- 初心者でも申し込みやすい
- 売り込み感が強すぎない
- ナチュラルで落ち着いている
- 現代的で洗練されている

## カラー方針

以下の色を中心に、全体のトーンを整えてください。

- 白
- ペールグリーン
- ベージュ
- くすみピンク
- やわらかいグレー
- 深すぎないブラウン系の文字色

黒が強すぎる場合は、やわらかいブラウンやチャコール系に調整してください。

## Codexで重点的に調整してほしいこと

### 1. 余白の調整

以下を確認してください。

- セクション間の余白が詰まりすぎていないか
- PC表示で窮屈に見えないか
- スマホ表示で余白が広すぎたり狭すぎたりしないか
- 見出しと本文の距離が自然か
- カード内の余白が整っているか
- CTA周りに十分な余白があるか

必要に応じて、Tailwindの `py`、`px`、`gap`、`space-y`、`max-w` などを調整してください。

### 2. レスポンシブ調整

PC、タブレット、スマホで自然に見えるようにしてください。

特に確認すること：

- スマホで横スクロールが発生していないか
- 文字が小さすぎないか
- CTAボタンが押しやすいサイズになっているか
- カードが崩れていないか
- 画像や装飾がはみ出していないか
- ファーストビューで目的がすぐ伝わるか
- セクションの順番が自然か

### 3. CTAの改善

LINE登録への導線を分かりやすくしてください。

確認すること：

- CTAボタンが目立っているか
- CTA文言が分かりやすいか
- 押したくなる安心感があるか
- ファーストビュー、中間、最終CTAに自然に配置されているか
- ボタンの色が背景に埋もれていないか
- hover時の反応が自然か

CTAは強引な売り込みではなく、安心して申し込める雰囲気にしてください。

### 4. 動き・アニメーションの追加

Webサイトとして自然な動きを追加してください。

ただし、派手すぎる演出は禁止です。

入れてほしい動き：

- ファーストビューのやわらかいフェードイン
- セクション表示時の自然なフェードアップ
- CTAボタンの軽いホバーアニメーション
- カードの軽い浮き上がり
- 背景装飾の控えめな動き
- スクロールしたくなる自然な流れ

避けてほしい動き：

- 激しいアニメーション
- 点滅
- 過剰なパララックス
- 読みにくくなる動き
- ヨガLPに合わない派手な演出
- 重くなる演出

Tailwind CSSで実装できる範囲の自然なアニメーションを優先してください。

必要であれば、CSSアニメーションや小さなユーティリティクラスを追加して構いません。

### 5. デザインの完成度調整

以下を確認して、ポートフォリオに載せられる見た目に整えてください。

- 見出しの強弱が分かりやすいか
- 本文が読みやすいか
- カードデザインが安っぽくないか
- 背景色が単調すぎないか
- 装飾が多すぎないか
- 女性向けヨガLPとして雰囲気が合っているか
- 余白を活かした上品な印象になっているか
- 全体のトーンに統一感があるか

### 6. コード品質の改善

以下を確認してください。

- buildエラーが出ない
- TypeScriptエラーが出ない
- 不要なimportがない
- 不要なconsole.logがない
- 使っていない変数がない
- classNameが極端に読みにくくなっていない
- コンポーネントが長すぎる場合は適度に分割する
- 同じようなUIが繰り返されている場合は配列化する
- 後からテキストを修正しやすい構造にする

### 7. アクセシビリティ

最低限、以下を確認してください。

- 見出し構造が不自然でない
- ボタンやリンクの意味が分かる
- 画像に必要なaltが入っている
- 文字色と背景色のコントラストが低すぎない
- スマホでタップしやすいサイズになっている

## 修正してほしくないこと

以下は大きく変えすぎないでください。

- LPのテーマ
- ターゲット
- LINE登録を目的とした導線
- 女性向けの上品でやさしい雰囲気
- 参考画像から読み取った基本的な方向性
